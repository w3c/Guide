The purpose of this page is to progressively list the resources useful when working on W3C projects using GitHub. The following links should help you find your way.

Repositories
If you are looking for a specific project that we maintain on GitHub, the best is to go straight to the list of repositories, which is searchable.
GitHub Help
Most of the issues people have using GitHub are in fact due to git. General git and GitHub questions will not be addressed here. GitHub's own help site linked above is a very helpful resource. You can also usually just cut and paste from Stack Overflow's git questions.
FAQ
Introductory-level questions that are commonly asked about using GitHub specifically in a W3C context.
The w3c.json file
This is a small metadata file that is recommended for GitHub repositories under the w3c organisation.
